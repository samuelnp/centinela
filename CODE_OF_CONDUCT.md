# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**, available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## Summary

- Treat everyone with respect and good faith.
- Welcome newcomers, reviewers, and contributors regardless of background or experience level.
- Disagree on ideas, not on people. Critique code and proposals — not the author.
- Assume positive intent; ask for clarification before reacting.
- No harassment, personal attacks, or discriminatory language.

## Scope

This Code of Conduct applies to all project spaces — issues, pull requests, discussions, code review comments, commit messages, and any direct communication tied to project work.

## Reporting

If you witness or experience unacceptable behavior, contact the project maintainer at **sanapo@gmail.com**. Reports are handled confidentially and in good faith. Maintainers reserve the right to remove comments, lock threads, or restrict participation when this Code is violated.

The canonical Contributor Covenant text linked above is the authoritative source for definitions and enforcement guidelines.
