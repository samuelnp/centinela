# Edge Cases: README Centinela Usage Refresh

- README must stay concise; detailed step-by-step prompts belong in `HOWTO.md`.
- HOWTO must show every enforced workflow step instead of implying code can start immediately.
- Acceptance guidance must mention executable acceptance tests and validation commands that run them.
- User-facing examples must mention `ux-ui-specialist` evidence and `mobileFirst: true`.
