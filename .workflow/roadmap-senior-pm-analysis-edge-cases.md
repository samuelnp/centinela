# Edge Cases: roadmap-senior-pm-analysis

## Covered

- Analysis JSON references unknown feature names.
- Analysis omits a roadmap feature.
- Analysis includes cyclic dependencies.
- Analysis role differs from `senior-product-manager`.
- Greenfield start blocks when roadmap exists but analysis artifacts are missing.

## Residual Risks

- Dependency analysis currently validates feature graph structure only.
- UX flow quality remains a human-reviewed concern in markdown narrative.
