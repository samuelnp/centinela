# Edge Cases — add-ux-ui-specialist-orchestration

- `surface: user-facing` must add UX/UI evidence only during the `code` step, not every step.
- Features without `surface: user-facing` must keep the previous strict role requirements.
- `ux-ui-specialist` evidence must fail when outputs are summaries or real files outside configured UI paths.
- UX evidence must still require edge cases, covering hierarchy, motion, skeletons, and responsive polish concerns.
