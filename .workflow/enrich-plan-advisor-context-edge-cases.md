# Edge Cases — enrich-plan-advisor-context

- Roadmap dependencies must be prioritized ahead of same-phase siblings when both exist.
- Missing roadmap analysis or quality artifacts must degrade gracefully to current-feature planning context.
- Related edge-case lessons should influence questions without dumping raw feature briefs or large prompt blocks.
- Advisor output must stay concise even when roadmap, sibling, and edge-case context are all available.
