# Edge Cases: docs-readme-bootstrap-tutorial

## Covered

- Users who run `centinela init` manually can follow the same flow as users who rely on setup hooks.
- Tutorial clarifies both generated roadmap artifacts: `ROADMAP.md` and `.workflow/roadmap.json`.
- Natural-language examples cover roadmap checks, feature start phrases, and continue-step phrases.
- Tutorial keeps command examples explicit for users who prefer direct CLI usage.

## Residual Risks

- Exact natural-language trigger behavior depends on the host agent integration quality.
- Teams with custom setup prompts may need slight phrasing adjustments in README examples.
