### 6. Correctness of the core promise — VERIFIED EMPIRICALLY (temp repo)
All run with `/tmp/cent-docgk` against a throwaway `/tmp` git repo; the real
repo/worktree were never mutated. Temp dir removed afterward.

  roadmap **ERROR**, exit **1**. `--fix` stripped the glyph, regenerated
  ROADMAP.md, re-diagnosis **OK** (exit 0). Second `--fix` →
  `roadmap.json` and `ROADMAP.md` **byte-identical** (idempotent).
- **Orphaned `*.json.tmp`** → flagged ERROR (exit 1); `--fix` removed it;

- **Exit semantics:** only-WARN → exit 0; any ERROR → exit 1. Confirmed both.
- **Worktree root resolution:** run from inside `.worktrees/<f>` resolved the

- **No active workflow required:** doctor ran with no active workflow present.
