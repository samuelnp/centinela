# Feature: README Bootstrap Tutorial

## Problem

New users can run `centinela init`, but README lacks one end-to-end tutorial showing
how setup interview, roadmap creation, and first feature kickoff fit together.

## Outcome

Add a practical README walkthrough that:

- demonstrates bootstrap flow from `init` to `centinela start <feature>`
- explains expected generated artifacts (`PROJECT.md`, `ROADMAP.md`, roadmap JSON,
  and feature briefs)
- clarifies natural-language prompts users can send to the agent to trigger
  centinela commands under the hood
