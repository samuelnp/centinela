# Feature: README Centinela Usage Refresh

## Problem

The README should reflect Centinela's current usage model, including the latest
workflow additions, agent integrations, setup guidance, roadmap behavior,
orchestration, validation, migration, and documentation generation. Users also
need a concrete example that shows how to collaborate with Centinela through an
agent to produce a small MVP landing page without bypassing the enforced steps.

## Outcome

Refresh `README.md` so users understand how to properly use Centinela today.
Add a focused `HOWTO.md` if the README would become too long, and link it from
the README. The guide should teach the correct `plan -> code -> tests ->
validate -> docs` flow for a small landing page MVP, including what to ask the
agent, when to approve step advancement, and which commands verify progress.
