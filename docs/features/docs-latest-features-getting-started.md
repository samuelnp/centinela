# Feature: Latest Features Docs Refresh

## Problem

The README and generated HTML docs no longer present Centinela's latest user-facing
capabilities as a cohesive story. New users can find individual commands, but they
do not get a current workflow-oriented getting-started path that teaches how to
bootstrap, navigate, and continue the Centinela process.

## Outcome

Refresh the project-facing documentation so users can quickly understand:

- the latest Centinela capabilities across setup, roadmap, migration, validation,
  and documentation flows
- how to get started with the enforced `plan -> code -> tests -> validate -> docs`
  workflow
- how Claude and OpenCode integrations surface setup prompts, workflow context,
  autostart, orchestration guidance, and status feedback
